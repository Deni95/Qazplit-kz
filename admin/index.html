Важные настройки Supabase

Важно

Для товаров нужен public bucket product-images и таблица products с полем images типа jsonb.

Для раздела “Наши работы” нужен public bucket works и таблица works с полями: media_url, media_type, title, created_at.

Если сохранение не работает, значит в Supabase нужно разрешить добавление файлов и строк для этих bucket/table.
